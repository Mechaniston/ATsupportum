Файл с настройками синхронизации данных

Если файл настроек синхронизации данных утерян, то для формирования нового файла необходимо:

  • перейти в первую программу
  • перейти в форму синхронизации данных
  • выбрать требуемую синхронизацию данных и открыть ее параметры
  • в командной панели формы следует выбрать пункт меню: "Все действия" –> "Получить настройки синхронизации данных для другой программы..."
