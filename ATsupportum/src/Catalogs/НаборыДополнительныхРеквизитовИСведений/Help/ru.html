Администратор системы может настроить наборы дополнительных свойств для определенных объектов системы. Способ хранения дополнительного свойства в информационной базе и место размещения на форме объекта зависит от типа свойства: дополнительный реквизит или сведение.

Предопределенные наборы соответствуют объектам конфигурации, в которых можно использовать дополнительные реквизиты и сведения. Добавление новых наборов невозможно.

Для набора выбранного в левой таблице можно добавить новые дополнительные свойства с помощью командной панели в правой таблице.

Примечание:

При пометке на удаление свойства в наборе, значения свойства сохраняется в объектах до тех пор, пока не будут очищены. При этом свойство, помеченное на удаление в наборе, будет отображаться на форме объекта или форме дополнительных сведений особым образом.
При необходимости можно снять пометку удаления.
