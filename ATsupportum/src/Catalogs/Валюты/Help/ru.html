Справочник «Валюты» предназначен для хранения списка используемых валют и их курсов.

Реквизиты:

  • Код числовой
    		•  Уникальный числовой код, служит для идентификации валюты при загрузке курсов из сети Интернет.
  • Код символьный
    		•  Краткое наименование, используется как основное представление валюты.
  • Наименование
    		•  Полное наименование.
  • Курс
    		•  Значение курса на текущую дату.

    Команды:

  • Подобрать из классификатора
    		•  Добавить валюту из Общероссийского классификатора валют (далее - ОКВ). Следующие реквизиты новой валюты будут заполнены автоматически: Наименование, Код символьный, Код числовой и флажок "Загружается из сети Интернет".
  • Создать
    		•  Добавить валюту вручную. Все реквизиты новой валюты необходимо будет заполнить вручную.
  • Загрузить курсы валют
    		•  Перейти к диалогу загрузки курсов валют с сайта РБК.

     

    См. также: Валюта, Загрузка курсов валют, Общероссийский классификатор валют, Общая информация о формах.
