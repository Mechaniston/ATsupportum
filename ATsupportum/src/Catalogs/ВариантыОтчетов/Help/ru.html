Редактирование, открытие и выбор вариантов отчетов.

В левой части формы расположено дерево разделов и групп командного интерфейса.
Перемещаясь по узлам дерева, список "Варианты отчетов" автоматически фильтруется по выбранному разделу или группе.
Варианты отчетов можно перемещать из текущего раздела/группы в другие при помощи перетаскивания. Если при перетаскивании зажать клавишу Ctrl, то вариант отчета будет добавлен в новую группу, а также останется в текущей группе.

Чтобы быстро найти нужный вариант отчета в списке "Варианты отчетов" также можно применять быстрые отборы по Наименованию и Автору. Для этого нужно указать искомые значения в соответствующих полях, расположенных над списком.

Колонки списка:

  • Наименование
    		•  Представление варианта отчета в списках.
  • Автор
    		•  Автор варианта отчета.

    Дополнительные колонки:

    (эти колонки можно включить через "Все функции" - "Настроить форму")

  • Картинка "Доступен всем пользователям"
    		•  Картинка видна - вариант отчета доступен всем пользователям,
    Картинка не видна - вариант отчета доступен только автору.
  • Флажок "Видимость по умолчанию в панели отчетов"
    		•  Значение по умолчанию флажка "Видимость" для всех пользователей в панелях отчетов.
  • Имя отчета
    		•  Имя, под которым отчет зарегистрирован в программе.
  • Отчет
    		•  Представление отчета.
  • Тип отчета
    		•  Внутренний, Дополнительный или Внешний.
    Варианты внешних отчетов не могут размещаться в панели отчетов.
  • Ключ варианта
    		•

    Имя предопределенного или уникальный идентификатор пользовательского варианта отчета.

    Команды:

  • Открыть
    		•  Открыть отчет для формирования и просмотра.
  • Разместить в разделах
    		•  Разместить несколько вариантов отчетов в разделах программы.
  • Сбросить настройки пользователей
    		•  Сбросить пользовательские настройки выбранных вариантов отчетов: видимость и быстрый доступ.
  • Сбросить настройки размещения
    		•  Восстановить исходные настройки выбранных вариантов отчетов: размещение в разделах программы, доступность и видимость по умолчанию.
    Может выполняться только для предопределенных вариантов отчетов.
