Форма позволяет указать настройки различных транспортов сообщений обмена, определенных в конфигурации, а также проверить правильность задания этих настроек.
Если создается новая запись из формы списка регистра сведений, то доступно поле "Узел информационной базы".
В поле "Узел информационной базы" указывается узел плана обмена, для которого выполняется настройка параметров транспорта.
На закладках формы задаются настройки отдельных видов транспортов.

Общими для всех являются следующие настройки:

  • "Вид транспорта сообщений обмена по умолчанию" – поле для задания вида транспорта, который будет использоваться по умолчанию при выполнении автоматического и интерактивного обмена данными.
  • "Пароль архива сообщения обмена" – поле для задания пароля для сжатия и распаковки архива сообщения обмена. Подразумевается, что пароль сжатия в базе источнике и пароль распаковки в базе приемнике должны быть одинаковыми для пары обменивающихся конфигураций.
